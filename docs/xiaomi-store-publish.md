## 📋 应用描述（商店用）

单点行人抓拍 - 离线值守抓拍应用

基于 Android CameraX 构建的离线智能行人识别与抓拍工具，完全本地运行，隐私安全有保障。

【核心功能】
• 实时监控：后置摄像头实时预览，AI智能匹配，检测到关注人员自动抓拍
• 关注底库：录入关注人员信息（姓名、性别、衣着、配饰、体态、照片），随时管理
• 抓拍记录：自动保存抓拍图片，支持缩略图浏览、大图查看、CSV导出
• 统计分析：今日抓拍数、目标分布柱状图、最近活动时间线
• 灵活设置：匹配阈值调节、去重间隔设置、最大记录数控制、震动提醒开关

【技术特点】
• 完全离线运行，数据存储在手机私有目录，不上传任何信息
• 多维度加权评分算法 + 滑动平均平滑 + 多帧确认机制
• 抓拍命中后5秒倒计时自动恢复值守
• Material Design 风格 UI

【使用场景】
• 安防监控：出入口值守，关注人员自动识别
• 寻人提示：设定关注目标，出现即提醒
• 活动管理：会场入口识别特定人员

## 🚀 发布流程

### 第一步：注册小米开发者账号

1. 访问 https://dev.mi.com
2. 使用小米账号登录（或注册新账号）
3. 选择「个人开发者」认证
4. 填写信息：
   - 开发者名称：按账号主体填写
   - 真实姓名：按账号主体填写
   - 身份证号码：******
   - 身份证扫描件：手持身份证正面照
   - 联系电话：******
5. 等待审核（1-3个工作日）

### 第二步：创建应用

1. 登录开发者站，点击「创建应用」
2. 填写：
   - 应用名称：单点行人抓拍
   - 操作系统：Android
   - 包名：com.localguard.pedestriancapture
3. 获取 appID、AppKey、AppSecret

### 第三步：上传 APK

1. 编译 Release 版本（需要签名）
2. 上传 APK 文件
3. 系统自动解析包名、版本等信息

### 第四步：填写应用信息

- 应用图标：512×512 PNG（已生成在 outputs/icon_512.png）
- 应用截图：至少3张（需截取运行画面）
- 应用描述：见上方
- 分类：工具
- 标签：安防、工具、AI

### 第五步：提交审核

- 审核周期：1-3个工作日
- 审核通过后自动上架

## 🔑 生成 Release 签名

```bash
# 生成签名密钥
keytool -genkey -v -keystore zhuapaq-release.keystore \
  -alias zhuapaq -keyalg RSA -keysize 2048 -validity 10000

# 配置 build.gradle.kts 签名信息后编译
./gradlew :app:assembleRelease
```
