## ADB 输入法解决方案

- **问题**：LatinIME 自动纠错把下划线 `_` 改成空格，文件名 `test_target.jpg` 被破坏
- **解决**：`adb shell input text "Test_001"` 直接注入字符（绕过键盘自动纠错）
- **注意**：`_` 会被 ADB 转为全角 `＿`，但内容仍正确保存
